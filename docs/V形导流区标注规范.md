# V形导流区严格版标注规范

## 1. 文档目的

本文档用于指导标注员对图像中的“V形导流区”进行严格标注。

本版本的核心目标是提高正样本判定精度，减少将负样本误判为正样本的情况。标注时应优先保证 precision，只有当正样本证据充分、直接、可复核时，才允许标注为 `true`。

本任务需要判断图像中是否存在同时满足以下条件的 V形导流区：

1. 必须是明确成立的导流区。
2. 导流区轮廓必须完整可见。
3. 整体形状必须为严格或近似 V形。
4. 不得是 Y形导流区、多分叉导流区或其他非 V形结构。

只要任一条件不满足，或存在明显不确定性，均应标注为 `false`。

## 2. 输出格式

格式要求：

1. 只能输出 JSON，不得输出额外解释、标题、注释或 markdown。
2. 字段固定为 `has_channelization`、`reason`。
3. `has_channelization` 必须是布尔值，只能为 `true` 或 `false`。
4. `reason` 必须是字符串。
5. 不得输出 `evidence`、拓扑类型、置信度等其他字段。

示例格式：

```json
{
  "has_channelization": true,
  "reason": "图像中部偏右可见由两侧实线边界和内部导流线构成的完整楔形导流区，该区域分隔主路与分流车道，整体轮廓为清晰的V形且未形成Y形多分叉结构。"
}
```

## 3. 核心任务定义

`has_channelization` 表示：图像中是否存在轮廓完整可见、导流关系明确、整体结构为严格或近似 V形，且不属于 Y形或多分叉结构的导流区。

本规范中的“V形导流区”是指：

图像中由道路边界线、导流线、导流岛、隔离设施、防撞设施、柔性立柱、护栏等元素共同构成的独立非通行区域或过渡区域。该区域对车辆或其他通行对象形成明确的分隔、引导、分流、收口或限制作用，并且其两侧主要边界形成清晰的收拢或发散夹角，整体轮廓呈严格或近似 V形、倒 V形、楔形或三角夹角结构。

请注意：

1. 本任务不是判断“是否存在普通导流区”，而是判断“是否存在严格意义上的 V形导流区”。
2. 只有导流区成立但形状不是 V形，应判 `false`。
3. 只有 V形轮廓但导流关系不成立，应判 `false`。
4. 导流区轮廓不完整、关键边界缺失或夹角关系不可稳定辨认，应判 `false`。
5. 整体更像 Y形、三叉形、多分叉形、条带状、矩形或不规则结构，应判 `false`。

## 4. 正样本判定标准

只有同时满足以下全部条件时，才可标注为 `has_channelization = true`：

1. 图像中存在明确导流区。
   该区域必须是独立的非通行区域或过渡区域，并对通行路径形成明确分隔、引导、分流、收口或限制作用。

2. 导流区由可见实体或标线共同构成。
   可包括道路边界线、导流线、导流岛、护栏、隔离设施、防撞设施、柔性立柱、标志牌等。单个设施、单条线或零散元素不足以支持正样本。

3. 导流区轮廓完整可见。
   需要能看清导流区的两侧主要边界、夹角位置、开口方向和整体范围。关键轮廓不得被严重遮挡、裁切、出画或模糊。

4. 整体形状为严格或近似 V形。
   两侧主要边界应形成稳定的收拢、发散或夹角关系，整体呈 V形、倒 V形、楔形或近似三角形。

5. 该区域不是 Y形或多分叉结构。
   不应存在明显“主干 + 两个分支”或三方向分流/汇流的整体组织关系。

常见正样本特征：

1. 两条主要边界形成清晰夹角，内部有斜向导流线，构成完整三角形或楔形导流区。
2. 导流岛、护栏、防撞设施或柔性立柱与道路边界共同形成完整 V形导流区域。
3. 导流区位于主路与分流车道之间，完整分隔两个通行方向，整体为单一 V形结构。
4. V形主体后方连接普通虚线车道线，但虚线不参与导流区轮廓闭合，也不形成三方向分流结构。

## 5. 负样本判定标准

出现以下任一情况时，应标注为 `has_channelization = false`：

1. 图中没有明确导流区。
2. 只有普通道路、车道线、路肩、隔离带、护栏、锥桶、标志牌等元素，未共同形成导流关系。
3. 存在导流区，但整体形状不是严格或近似 V形。
4. 存在导流区，但轮廓不完整，关键边界被遮挡、裁切、出画或难以辨认。
5. 只有局部像 V形，但无法确认完整轮廓和导流作用。
6. 整体更像 Y形、三叉形、多分叉形、条带状、矩形、路肩缓冲区或不规则过渡区。
7. 图像模糊、距离过远、角度受限或遮挡严重，无法稳定判断导流区和 V形结构同时成立。
8. 需要依赖推测、道路常识或场外信息才能判断为 V形导流区。

常见负样本特征：

1. 普通车道虚线、实线或道路边缘线单独出现，不构成独立导流区。
2. 设施或标线形成了导流关系，但区域较长且更像条带状缓冲区，不呈 V形。
3. 导流区域存在多个分支方向，整体更接近 Y形或三叉分流结构。
4. 画面只显示导流区的一部分，无法看清夹角、边界和完整轮廓。
5. 导流区被车辆、护栏、画面边缘或阴影遮挡，关键结构不可复核。

## 6. V形与 Y形的区分

本版本必须严格区分 V形导流区与 Y形导流区。Y形导流区、多分叉导流区不属于本任务正样本。

### 6.1 V形导流区

V形导流区的核心特征是：

1. 导流区主体是一个单一的夹角区域。
2. 两侧主要边界形成收拢或发散关系。
3. 整体轮廓为 V形、倒 V形、楔形或近似三角形。
4. 通行组织主要体现为两侧路径之间的分隔、收口或分流。
5. 不存在明显中心节点向三方向展开的结构。

### 6.2 Y形导流区

出现以下特征时，应视为 Y形或多分叉结构，并判 `false`：

1. 整体呈现“主干 + 两个分支”的通行组织。
2. 导流区连接或影响三个方向的通行路径。
3. 有一条连续实线、隔离设施、护栏或道路边界作为主干，并与两个分支共同构成三叉结构。
4. 画面中的导流设施承担三方向分流、汇流或分叉作用。
5. 局部虽然有 V形夹角，但整体道路组织更接近 Y形。

### 6.3 V形加虚线的处理

若导流区主体轮廓为完整 V形，后方仅连接普通虚线车道线，且该虚线只是车道延续或车道分界提示，不参与导流区边界闭合，也不形成实质三方向分流关系，则仍可判为 V形导流区。

判定为 V形的条件：

1. V形主体轮廓完整可见。
2. 两侧主要边界构成明确夹角。
3. 虚线不是导流区实体边界。
4. 虚线没有与 V形主体共同形成“主干 + 两个分支”的 Y形结构。

若后接的是连续实线、护栏、隔离设施或明确道路边界，并与导流区共同形成三方向通行组织，则应判为 Y形或多分叉结构，标注为 `false`。

## 7. 边界样本处理原则

对于难以判断的图像，请遵循以下原则：

1. 宁可漏判边界正样本，也不要在依据不充分时判 `true`。
2. 只要导流区是否成立存在疑问，应判 `false`。
3. 只要 V形轮廓是否完整存在疑问，应判 `false`。
4. 只要 V形与 Y形之间难以区分，应判 `false`。
5. 只要需要依靠推测补全被遮挡或出画的关键边界，应判 `false`。
6. 对低清、远景、强遮挡、强透视样本，只有在完整 V形轮廓仍清晰可复核时，才可判 `true`。

本版本不鼓励对不完整样本作宽松正样本判定。上一版本中“局部遮挡但仍可判断”的样本，在本版本中应重新检查：若关键轮廓不完整或不可复核，应判 `false`。

## 8. 最小判定流程

标注时请按以下顺序检查：

1. 是否存在候选导流区。
   若没有独立非通行区域或过渡区域，直接判 `false`。

2. 导流关系是否成立。
   若该区域没有对通行路径形成明确分隔、引导、分流、收口或限制，直接判 `false`。

3. 轮廓是否完整可见。
   若两侧边界、夹角、开口方向或整体范围不可完整辨认，直接判 `false`。

4. 是否为严格或近似 V形。
   若整体不是 V形、倒 V形、楔形或近似三角夹角结构，直接判 `false`。

5. 是否属于 Y形或多分叉结构。
   若存在三方向分流/汇流、主干加分支、复杂分叉关系，直接判 `false`。

只有上述五步全部通过，才能判 `true`。

## 9. reason 字段书写要求

`reason` 用于说明“为什么这样判断”，必须基于图像中可见内容，不得凭空推测场外信息。

### 9.1 判 true 时的要求

若标注为 `true`，`reason` 必须同时说明：

1. 导流区位于图像中的位置。
2. 导流区由哪些可见边界、标线或设施构成。
3. 为什么导流关系成立。
4. 为什么轮廓完整可见。
5. 为什么整体为 V形，且不是 Y形或多分叉结构。

推荐控制在 1 到 3 句内，语言简洁、具体、可复核。

### 9.2 判 false 时的要求

若标注为 `false`，`reason` 应说明最关键的否定原因，例如：

1. 未形成明确导流区。
2. 导流区轮廓不完整或关键边界不可见。
3. 整体不是 V形。
4. 更接近 Y形、三叉形或多分叉结构。
5. 画面模糊、遮挡、距离远或角度受限，无法稳定判断。
6. 仅有普通车道线、路肩、护栏或单个设施，不足以支持 V形导流区成立。

### 9.3 不推荐写法

以下写法不建议使用：

1. “这是V形导流区。”
2. “图中有导流。”
3. “看起来像V形。”
4. “有线和护栏，所以是导流区。”
5. “应该是V形。”

上述写法信息量不足，不能反映真实判定依据。

## 10. 标注注意事项

1. 不要因为出现单个设施、单条斜线、单侧边界或局部夹角就判 `true`。
2. 不要把普通车道线、路肩边缘、匝道边缘或缓冲带误判为 V形导流区。
3. 不要把 Y形、三叉形、多分叉导流区误判为 V形导流区。
4. 不要把“可能是”“猜测是”“大概是”作为判定依据。
5. 如果判断依据不充分，请直接在 `reason` 中说明证据不足，并标注为 `false`。
6. 如果图中存在多个道路元素，请优先描述对 V形导流区判定最关键的结构，不要罗列无关内容。
7. 本版本以严格筛选正样本为目标，边界样本默认按负样本处理。

## 11. 质检建议

质检时建议重点关注以下问题：

1. 是否把普通导流区误判为 V形导流区。
2. 是否把 Y形或多分叉结构误判为 V形导流区。
3. 是否把轮廓不完整、遮挡严重、出画或模糊样本误判为正样本。
4. 是否因为单条线、单个设施或局部夹角误判为正样本。
5. `reason` 是否同时解释了导流区成立、轮廓完整、V形结构和非 Y形依据。
6. 负样本 reason 是否明确指出了不满足条件的具体原因。

## 12. 最终执行要求

标注员在实际执行中，请始终遵循以下原则：

1. 先判断是否存在明确导流区。
2. 再判断导流区轮廓是否完整可见。
3. 再判断整体是否为严格或近似 V形。
4. 最后排除 Y形、多分叉形和其他非 V形结构。
5. 任一关键条件不满足或证据不足，均判 `false`。
6. 本版本优先保证 precision，避免将负样本误判为正样本。

如无特殊说明，后续严格版 V形导流区标注工作均以本规范为准。
